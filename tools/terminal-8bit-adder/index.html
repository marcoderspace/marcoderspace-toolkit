> terminal 8-bit adder

simple interactive mock cpu that really adds two 8-bit numbers and shows each step.

legend / how to use 1) enter a and b. 2) click load to place values into the simulated registers. 3) click step to move one micro-step at a time. 4) click run to execute the whole addition automatically. 5) result appears in register c / out. micro-steps: - fetch a → move a into alu input - fetch b → move b into alu input - exec add → a + b - write c → write result to output register - halt → stop notes: - this is 8-bit arithmetic, so max visible value is 255. - if the sum is above 255, carry becomes 1 and result wraps modulo 256.
