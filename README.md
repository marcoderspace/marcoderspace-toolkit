# Marcoderspace Toolkit

A collection of browser-native code and experiments.

This repository is structured as a set of independent, self-contained units built with a focus on clarity and simplicity.

## Structure

The repository is organized into separate directories.

Each directory contains its own implementation and documentation.

## Approach

- Keep things minimal
- Keep things readable
- Avoid unnecessary abstraction
- Build in a way that can be inspected directly
